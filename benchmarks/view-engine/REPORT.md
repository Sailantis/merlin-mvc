# Benchmark Report

This report summarizes the benchmark comparing view engines: Clarity, Native, Twig, Plates, Blade.

Run command used (example):

```bash
php -d opcache.enable_cli=1 benchmarks/run.php --engines=clarity,native,twig,plates,blade --iterations=10000 --out=last_run
```

Summary (from example run):

- Clarity: fastest steady-state for compiled templates; low memory usage.
- Native: very fast with OPcache for included PHP templates.
- Plates: flexible and reasonably fast; suitable when using PHP templates with a template system.
- Twig: richer feature set, higher warm/compile time; use cache + OPcache for best steady-state.
- Blade: featureful but slower steady-state than Clarity/Native; has warm compile overhead.

Interpretation & recommendations:

- For raw throughput and low runtime overhead, prefer `Clarity` or `Native`.
- For a good balance between ergonomics and performance, `Plates` is a strong choice.
- If you need a full-featured template language with many extensions, `Twig`/`Blade` are options but accept additional cost.

Files generated by the harness:

- `benchmarks/last_run.json` — detailed per-engine metrics.
- `benchmarks/last_run.csv` — CSV table for spreadsheets.
